# 🚀 FoundHer AI

### The AI Co-Founder for Women Entrepreneurs

FoundHer AI is an AI-powered startup companion designed specifically for women entrepreneurs. The platform helps founders transform ideas into launch-ready startups through AI guidance, mentorship, startup planning, networking, and execution support.

> **Mission:** Empower every woman with an idea to become a founder.

# 📂 Project Structure

| Directory / File | Description |
| :--- | :--- |
| **`public/`** | Static assets served directly to the browser (e.g., favicons, metadata). |
| **`src/components/`**| Reusable UI components (buttons, inputs, cards) powered by Shadcn UI. |
| **`src/hooks/`** | Custom React hooks containing reusable stateful logic. |
| **`src/integrations/`** | Third-party service configurations and API integrations. |
| **`src/lib/`** | Utility helper functions and global initializers (e.g., Supabase client). |
| **`src/pages/`** | Application views/screens corresponding to your router paths. |
| **`src/App.tsx`** | Root component managing routing and global context providers. |
| **`src/main.tsx`** | Application entry point where React anchors to the DOM. |
| **`supabase/`** | Backend directory containing database migrations and Edge Functions. |
| **`components.json`** | Configuration file mapping your Shadcn UI components setup. |
| **`vite.config.ts`** | Core bundler configuration file for the Vite development environment. |

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/techiee-jhalak/FoundHer-AI.git
cd FoundHer-AI
```

## Install Dependencies

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```

## Build Project

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```
